# Starwars Planets Search

Você vai desenvolver uma lista com filtros de planetas do universo de Star Wars usando Context API e Hooks para controlar os estados globais.

## Projeto

![Print do Projeto](https://i.imgur.com/xUfN36F.png)

## Executar Localmente

Clone o projeto 

```bash
  git clone git@github.com:brunokye/trybe-15-starwars-planets-search.git
```

Entre no diretório

```bash
  cd trybe-15-starwars-planets-search/
```

Instale as dependências

```bash
  npm install
```

Inicie o projeto

```bash
  npm run start
```

## Executar os Testes

Use o comando abaixo dentro do diretório do projeto

```bash
  npm run test
```
